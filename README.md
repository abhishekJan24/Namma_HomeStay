\# Namma\_HomeStay



Namma\_HomeStay is an Android-based Homestay Management Application developed using Kotlin, MVVM Architecture, Firebase Firestore, and Firebase Storage.



\## Features



\- Homestay property management

\- Upload room, toilet, and farm images

\- Daily food menu management

\- Customer inquiry handling

\- Local guide information

\- Real-time Firebase synchronization

\- User-friendly Android UI



\## Technologies Used



\- Kotlin

\- Android Studio

\- MVVM Architecture

\- Firebase Firestore

\- Firebase Storage

\- XML Layouts

\- RecyclerView



\## Installation



1\. Clone repository



```bash

git clone https://github.com/abhishekJan24/Namma\_HomeStay.git

```



2\. Open in Android Studio



3\. Sync Gradle



4\. Run on emulator or Android device
